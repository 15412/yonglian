Company profile / 公司简介

永联织造是世界前三的童装生产商，成立于1972年在香港成立，由阮氏家族和（王）氏家族拥有公司100% 的股权。1986年在大陆先后于东莞和广东韶关开设永联织造厂，拥有32间车间，总员工数超过10000人，主要 生产婴儿及儿童服装，年产量约1.2亿件。公司拥有超过40年的生产和出口经营优势，产品出口到美国，英国， 欧洲，加拿大，澳大利亚和亚洲各地，并获得了Carter’s 、Tesco、Nike、Target、Walmart等公司的指定 生产商。公司以专注生产婴儿、儿童服装为核心，注重关爱儿童健康成长，成为儿童服装市场的前驱和专家。
